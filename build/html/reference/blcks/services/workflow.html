WorkflowManager¶

Pentium Workflow service class

class blcks.services.workflow.WorkflowManager(cred, event, config)

WorkflowManager: Workflow management base class since class might inheritant from multiple base classes, all the functions need a prefix or postfix to distinguish the difference

Raises

getWorkflows(sort='created', order=<OrderBy.Desc: 'desc'>, offset=0, limit=0, search=None)

Get all workflows

Parameters

  • sort (str) – which elements you want to sort

  • order (OrderBy) – sorting order, now support OrderBy.Desc and OrderBy.Asc

  • offset (int) – the offset of the object array index, default is 0 if this parameter is not presented

  • limit (int) – the amount of object will be returned from the specified offset, default is 10 if its’ not presented

  • search (str) – the search tags user input
