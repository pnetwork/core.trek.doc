Parameters

  • secrets (bool) -- to get encrypted ssh password and keyPair content

  • HostIds (str) -- to get specifict host with ids arary string, ex: s-abc,s-bbb,s-ccc

  • HostId (str) -- to get specifict host with id

  • limit (int) -- The amount of object will be returned from the specified offset, default is 10 if its’ not presented

  • search (str) -- A search string that specify the qualification condition

  • order (OrderBy) -- The order of Sort

  • sort (str) -- which elements you want to sort

  • hostUsage (HostUsage) -- to identify server usage EX. PROXY NORMAL

  • isCloud (bool) -- to identify the source of host is from cloud or not
