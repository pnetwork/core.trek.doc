ChannelManager¶

Pentium Channel service class

class blcks.services.channel.ChannelManager(cred, event, config)

ChannelManager: Channel management base class since class might inheritant from multiple base classes, all the functions need a prefix or postfix to distinguish the difference

getChatbotChannels(sort='created', order=<OrderBy.Desc: 'desc'>, offset=0, limit=0, search=None)

do get message to specified channel :type sort: str :param sort: which elements you want to sort :type order: OrderBy :param order: sorting order, now support OrderBy.Desc and OrderBy.Asc :type offset: int :param offset: the offset of the object array index, default is 0 if this parameter is not presented :type limit: int :param limit: the amount of object will be returned from the specified offset, default is 0 which means get all if its’ not presented :type search: str :param search: the search tags user input
