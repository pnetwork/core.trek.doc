initenv¶

Usage: [OPTIONS] [ACTIONS]...

Create and start containers of base and blcks

Options:
-p, --path TEXT

Marvin workflow project path

--no-overwrite

Not re-launch container when it already exist.

-v, --verbose

verbose log

--help

Show this message and exit.

Example¶

$ trek initenv
Starting environment...
starting trek-router...
Done
