Trek Shell: Deploy Packed Packages to Marvin¶

Purpose:

發佈已 packed 的 Shell 專案至 Marvin 平台。

Usage:

Step 1. 當 Marvin 平台上已有同名的腳本、工作流程存在時，是否要覆蓋：

  • Overwrite existing scripts on Marvin

    填入 Y or N，大小寫不拘。若為 Y 表示要覆蓋 Marvin 平台同名腳本、工作流程。
