Trek: Logs¶

Purpose:

顯示執行日誌，支援查看全部日誌或是查看指定腳本 id 日誌。

Usage:

  • Please enter script id

    請輸入想查看 log 的腳本 id 或 *，如：

    • * : 列出所有 log 。

    • notification : 列出 notification 腳本的 log。
