deployblcks¶

Usage: [OPTIONS]

Deploy blcks project

Options:
-p, --path TEXT

Blcks project path

--projects TEXT

Assign projects

--autopack

Automatic pack all.

--autobuildpush

Automatic build and push all if need.

-y, --yes

Automatic yes to prompts.

-v, --verbose

verbose log

--help

Show this message and exit.
