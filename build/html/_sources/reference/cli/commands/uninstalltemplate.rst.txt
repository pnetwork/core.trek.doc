
*****************
uninstalltemplate
*****************

Usage:  [OPTIONS]

  Install template

Options:
  -c, --category TEXT  Template category
  -n, --name TEXT      Template name
  -v, --verbose        verbose log
  --help               Show this message and exit.


uninstall custom template

Example
-------

.. code:: console

    $ trek uninstalltemplate -n myscript -c script
    Uninstall template [myscript] from script
