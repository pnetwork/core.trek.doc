
*****************
createproject
*****************

Usage:  [OPTIONS] NAME

  Create a trek CLI project

Options:
  -f, --force    force creating project
  -y, --yes      Automatic yes to prompts
  --example      Create from example template
  -v, --verbose  verbose log
  --help         Show this message and exit.




Example
-------

.. code:: console

    $ trek createproject hello_project
    Project [hello_project] creating...
    This utility will walk you through creating a config.json file.
    Marvin url []:
    Marvin JWT []:
    Marvin secret []:
    Done
