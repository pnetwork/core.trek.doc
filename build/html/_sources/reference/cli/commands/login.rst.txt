
*****************
login
*****************

Usage:  [OPTIONS]

  Log in to Marvin

Options:
  --global       Use config from global
  -v, --verbose  verbose log
  --help         Show this message and exit.


Login Marvin to get token and write into ``config.json``.

Example
-------

.. code:: console

    $ trek version
    Username: your-name
    Password: your-password
    Marvin url [https://trek.pentium.network/]:
    Login to Marvin: [https://trek.pentium.network/]
