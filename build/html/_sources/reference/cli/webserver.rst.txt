
*****************
webserver
*****************

Usage:  [OPTIONS]

  Start a trek CLI management portal

Options:
  -p, --port TEXT  The port on which to run the server
  --host TEXT      Set the hostname on which to run the web server
  --debug          Use debug mode
  --help           Show this message and exit.




Example
-------

.. code:: console

    $ trek webserver
    * Serving Flask app "pnmflow.www.app" (lazy loading)
    * Environment: development
    * Debug mode: off
    * Running on http://127.0.0.1:5050/ (Press CTRL+C to quit)
