*****************************************************************
Trek: List Remote Scripts
*****************************************************************

Purpose:

  | 列出遠端機器上的腳本及版號。  
  | 遠端機器位置於 *config.json* 中 ``script_repository`` 欄位設定。


Usage:


- :Please enter script id: 請輸入想查看的腳本 id 或 ``*``，如：
    
    - ``*`` : 列出遠端機器上的所有可用腳本與版號。
    - ``notification`` : 列出遠端機器上 notification 腳本可用的版號。
