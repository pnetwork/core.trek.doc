
*****************
shutdownenv
*****************

Usage:  [OPTIONS] [ACTIONS]...

  Stop and remove containers of base and blcks

Options:
  -p, --path TEXT  Marvin workflow project path
  -a, --all        Down all containers
  -v, --verbose    verbose log
  --help           Show this message and exit.




Example
-------

.. code:: console

    $ trek shutdownenv
    Closing environment...
    closing trek-router...
    closing all other containers...
    Done
